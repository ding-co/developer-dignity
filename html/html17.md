## [HTML 제대로 배워 볼래?] Ep#17-주석처리

### _주석 처리_

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <!-- 아래 p 태그는 어떤 목적으로 작성했습니다. -->
    <!-- 주석작성 -->
    <!-- 주석 -->
    <!-- <p>문장입니다.</p> -->
    <p>문장입니다2.</p>
    <script></script>
  </body>
</html>
```

#

### [Note]

- 주석; 낱말이나 문자의 뜻을 쉽게 풀이 함
  1. 코드에 대한 자세한 설명 다는 용도 (충분한 설명)
  2. 특정 코드 잠시 실행 막기 위한 용도
- 누가 봐도 이해 가능한 주석 작성하는 것이 매우 중요!
- Ctrl + / [슬래시]
- 드래그해서 여러 줄 주석 처리
