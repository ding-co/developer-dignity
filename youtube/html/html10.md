## [HTML 제대로 배워 볼래?] Ep#10-하이퍼링크 태그

### _링크 태그_

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <a href="http://www.youtube.com/c/개발자의품격">개발자의 품격</a>
    <script></script>
  </body>
</html>
```

#

### [Note]

- a 태그 기능
  1. 특정 웹사이트(웹페이지)로 이동
  2. 동일한 페이지 내에서 스크롤 대신 특정 콘텐츠가 있는 위치로 이동
  3. 자바스크립트 함수 호출
- href는 attribute (속성)
