## [HTML 제대로 배워 볼래?] Ep#11-img 태그

### _img 태그_

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <img src="이미지주소" style="width: 200px; height: 200px" />
    <script></script>
  </body>
</html>
```

#

### [Note]

- 이미지 태그는 시작/종료 태그가 없음
  - 단일 태그임
- 해상도; 픽셀(점)
- 절대경로 / 상대경로
