## [HTML 제대로 배워 볼래?] Ep#05-HTML 기본구조 및 실행

### _HTML 기본 구조 및 실행_

```html
<!-- 가장 최신 HTML 버전 사용 -->
<!DOCTYPE html>
<html>
  <head>
    <title>페이지 제목</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
```

#

### [Note]

- localhost: 로컬 PC (아직 도메인 X)
- 도메인은 필요시 구매
  - 웹 호스팅 서버 이용해서 연결, 배포
- 라이브 서버 익스텐션
